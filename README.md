# SOCKS
SOCKS Protocol Library in Rust

## Example
SOCKS5 simple server
```shell
cargo run --example transparent
```

## Usage
```toml
[dependencies]
socks-lib = { version = "*" }
```

## Protocol Specification
[RFC 1928: SOCKS Protocol Version 5](https://datatracker.ietf.org/doc/html/rfc1928)
